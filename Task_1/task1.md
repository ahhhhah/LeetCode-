# 20. 有效的括号

## 问题描述
给定一个只包括 '('，')'，'{'，'}'，'['，']' 的字符串，判断字符串是否有效。

有效字符串需满足：

- 左括号必须用相同类型的右括号闭合。
- 左括号必须以正确的顺序闭合。
注意空字符串可被认为是有效字符串。

## 示例
输入: "()"
输出: true

输入: "(]"
输出: false

## 解析

用数据结构中的栈做括号匹配,唯一需要特别注意的地方:**所谓有效的括号一定是成对出现的**.
```c++
str.length() % 2 == 0
```
