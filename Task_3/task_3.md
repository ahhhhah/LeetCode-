# 合并K个排序链表

## 问题描述
合并 k 个排序链表，返回合并后的排序链表。请分析和描述算法的复杂度。

## 示例
输入:
[
  1->4->5,
  1->3->4,
  2->6
]
输出: 1->1->2->3->4->4->5->6

## 解析
留下了只会暴力求解的泪水....

思路:
- 将给出的所有链表中的节点值弹出
- 添加至新的链表中
- 将这个新链表排序
- 重新构成节点

**注意**
使用了辅助节点(头结点),返回时应为头结点的下一个节点开始的链表
